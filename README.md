## Usage

![image](./assets/image.png)

### Build the image
```sh
docker build -t caetun .
```

### Start docker compose
```sh
docker compose up -d
```

### Test
```sh
docker exec -it caetun-client1 ping 10.8.0.3
docker exec -it caetun-client2 ping 10.8.0.2
```
